Quick Rules

SpacingAlways Fibonacci token, never arbitrary px
TypeAlways golden ratio scale token
FontsExactly 3 per project — display, body, accent
RadiiPetal curves on interactive, clean edges on structural
ColorsNo pure #FFF or #000. No drop shadows.
MotionCSS transitions first. Respect prefers-reduced-motion.
LayoutGolden ratio (61.8/38.2) governs all splits
